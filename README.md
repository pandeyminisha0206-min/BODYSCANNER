# 🧍 Body Scanner (Pose Detection using OpenCV)

A real-time **body scanning system** built with Python that detects human body joints and draws a skeletal structure using a webcam feed.

---

## 📖 Description
This project uses **OpenCV** and **MediaPipe** to perform real-time human pose detection. When a person stands in front of the camera, the system identifies key body landmarks such as arms, legs, and joints, and connects them to form a live skeleton overlay.

---

## ✨ Features
- 🎥 Real-time webcam body detection  
- 🦴 Detects body joints and landmarks  
- 🔗 Connects joints to form skeleton structure  
- ⚡ Fast and lightweight  
- 💻 Simple and easy to run  

---

## 🛠️ Tech Stack
- Python  
- OpenCV  
- MediaPipe

## ⚙️ Installation

```bash
git clone https://github.com/your-username/body-scanner.git
cd body-scanner
pip install opencv-python mediapipe
```

▶️ How to Run 

```python body_scanner.py```

- Stand in front of your webcam
- The system will detect your body
- Joints and skeleton will appear on screen


📸 Output
- Live video feed
- Body joints highlighted
- Skeleton drawn over body
